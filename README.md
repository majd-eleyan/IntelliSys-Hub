## مميزات وتفاصيل الموقع

* **🗂️ تنظيم متكامل حسب السنوات والفصول:** المواد مقسمة بوضوح بناءً على سنوات الدراسة الأربعة (أولى، ثانية، ثالثة، رابعة)، وداخل كل سنة ستجد المواد مرتبة حسب الفصل الدراسي الأول والثاني مع تفاصيل الساعات المعتمدة لكل مساق (مثل: لغة برمجة 1، تصميم المنطق الرقمي، رياضيات منفصلة.. إلخ).

* **📂 مستكشف الملفات الذكي:** نظام منظم يتيح لك الوصول المباشر إلى (الكتب، الشباتر، السلايدات، الملخصات، ونماذج الامتحانات) الخاصة بكل مساق عبر زر "عرض الملفات" دون أي تعقيد.

* **💡 نصيحة مجرّب:** دليل إرشادي داخل كل مادة يحتوي على نصائح سريعة ومفاتيح أساسية تلخص طبيعة المساق وكيفية التعامل معه واجتيازه بنجاح.

* **📞 دليل الخدمات والأرقام الجامعية:** قاعة بيانات مصغرة تضم أرقام التواصل المباشر مع الإدارات المختلفة بالجامعة (عميد الكلية، الإدارة المالية، شؤون الطلاب، القبول والتسجيل) لتسهيل المعاملات الطلابية.

* **🤖 ربط ذكي مع المنصات الطلابية:** يضم الموقع وصولاً سريعاً لـ:
    * **مودل بوت (Moodle Bot):** للحصول على إشعارات المودل أولاً بأول.
    * **مجموعة الواتساب:** للانضمام إلى المجموعة للتخصص والتواصل مع الزملاء.

* **🌙 واجهة (Dark Mode):** تصميم عصري مريح للعين يدعم الوضع الليلي لتسهيل الدراسة لساعات طويلة.

## لتقنيات المستخدمة

تم بناء الموقع بالاعتماد على تقنيات الويب الأساسية لضمان السرعة، الخفة، والاستجابة العالية:
* **HTML5** (لبناء هيكل الموقع)
* **CSS3** (لتصميم الواجهات والـ Dark Mode والأنماط البصرية)
* **JavaScript** (لإدارة مستكشف الملفات، عمليات البحث، وتصفح المساقات)
* **GitHub Pages** (للاستضافة السريعة والمجانية للموقع)
